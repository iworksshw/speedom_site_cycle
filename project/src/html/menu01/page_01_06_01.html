활동지수는 아래 활동 항목에 따라 제공합니다.

* 단위는 P(포인트)라고 합니다.

가입

  • 신규회원가입 시 50,000P를 제공
  • 기존회원은 홈페이지 오픈 시 일괄 100,000P를 제공
  • 신규회원가입 후 첫 로그인 시 50,000P를 일일 1회 제공
  • 매일 로그인 시 10,000P를 일일 1회 제공

참여

  • 예상과 후기의 게시글 작성 시 10,000P를 일일 최대 5회 제공
  • 예상과 후기의 댓글 작성 시 5,000P를 일일 최대 10회 제공
  • 예상과 후기의 게시글 추천 시 1,000P를 일일 최대 10회 제공
  • 포인트 사용 규정 활동지수 포인트는 모의베팅 사용 시 적립된 포인트로 사용이 가능합니다.
  • 포인트 유효 기간 경륜경정 홈페이지에서 진행되는 활동지수이벤트를 통해 당첨자 선정 후 모든 회원은 일괄 초기화되며 활동에 따라 재적립 됩니다.
  • 포인트 소멸 기준 회원탈퇴 시 보유한 활동지수는 즉시 소멸됩니다.
  • 포인트 지급 정지 부정행위/경고/신고 대상 회원은 활동지수 지급이 해제 시까지 정지됩니다.
  • 포인트 확인 및 관리 회원은 활동지수현황에서 활동지수 내역을 실시간으로 확인합니다.
