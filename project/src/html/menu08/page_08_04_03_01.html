광명스피돔 위치

경기도 광명시 광명로 721(광명동)

자가용 이용

성남 및 분당지역 수도권 제1순환고속도로 이용

청계요금소에서 안현분기점까지 직진 -> 제2경인국도 광명방향으로 변경(안현분기점) -> 광명IC에서 나와 광명시청방면으로 우회전 -> 첫째 사거리(능촌)에서 직진 -> 둘째 사거리(밤일)에서 직진 -> 왼쪽으로 광명돔경륜장이 보이는 셋째 사거리(광남)에서 U턴후 광명문으로 진입(광명IC -> 광명스피돔 : 3.0km)

강북(마포, 신촌방면)

성산대교 -> 서부간선도로 진입지점에서 목동방향으로 고가를 타고 우회전 -> 고가에서 내려온후 좌회전 -> 길 끝에서 좌회전 -> 길끝에서 우회전(목동병원을 끼고) -> 직진(광명 이정표 있음) -> 광명시청 방향으로 우회전 -> 광명시청 삼거리에서 우회전 -> 광명사거리역에서 좌회전 -> 광명돔경륜장(1.6km)

서울숲, 성수대교 북단방향

강변북로(일산방향) -> 양화대교 횡단 -> 경인고속도로 입구(우회전) -> 서부간선도로 진입광명대교 진입(우회전) -> 현충로 삼거리(철산주공 8단지 앞, 좌회전) -> 광명시청아프 삼거리(광명시민회관앞, 우회전) -> 광명사거리(광명IC방면으로 좌회전) -> 광명스피돔 도착

사당방면

봉천사거리에서 서울대방향(좌회전) -> 서울대 지나서 미림여고방향 좌회전 -> 경인교대 반대방향으로 제2경인국도 -> 광명IC도착광명IC에서 나와 광명시청방면으로 우회전 -> 첫째 사거리(능 촌)에서 직진 -> 둘째 사거리(밤일)에서 직진 -> 왼쪽으로 광명돔경륜장이 보이는 셋째 사거리(광남)에서 U턴후 광명문으로 진입(광명IC -> 광명스피돔 : 3.0km)

봉촌, 신림동 방향

봉천사거리에서 서울대방향(좌회전) -> 서울대 지나서 미림여고방향 좌회전 -> 경인교대 반대방향으로 제2경인국도 -> 광명IC도착광명IC에서 나와 광명시청방면으로 우회전 -> 첫째 사거리(능 촌)에서 직진 -> 둘째 사거리(밤일)에서 직진 -> 왼쪽으로 광명돔경륜장이 보이는 셋째 사거리(광남)에서 U턴후 광명문으로 진입(광명IC -> 광명스피돔 : 3.0km)

영등포, 구로지역

남부수환도로 오류IC ->> 오류동방향 -> 고가차도 통과후 오른쪽 교통안전공단(자동차검사소)를 지나 200m 통과 후 광명방향 좌회전 -> 천왕역 -> 광명교에서 우회전 -> 광남사거리
