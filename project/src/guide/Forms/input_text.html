Input Text
- input text, password 폼입니다.
- 기본 폼이며 검색, 그리드, 테이블에 동일한 폼을 사용합니다
- 가능한 .frmGroup 내부에서 사용합니다.
- .iptText는 인풋 클래스이며 가로는 기본 100%입니다 .lg, .sm으로 크기를 조절합니다.
- .frmBox는 가로 너비 및 gap을 담당합니다.
Input Text 너비
- input text, password 및 select의 너비 가이드입니다.
- .frmBox에 너비 값을 적용합니다. 기본 너비는 자동 분할입니다. (100% / 인풋 갯수)
- .frmTini : 제일 작은 크기 (정의되지 않았습니다).
- .frmXs : 많이 작은 크기 (정의되지 않았습니다).
- .frmSm : 작은 크기 (정의되지 않았습니다).
- .frmMd : 보통크기 : 300px, 모바일 : 100%
- .frmLg : 큰 크기 (정의되지 않았습니다).
- .frmXL : 매우 크기 (정의되지 않았습니다).
